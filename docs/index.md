# 笔记站点结构总览

> 本首页从“整个仓库”视角，简要说明笔记站点的结构组成与用途，便于快速定位与浏览。

---

## 顶层结构（可跳转）

- [`00-Inbox/`](../00-Inbox/)：临时收集与待整理内容。
- [`01-Notes/`](../01-Notes/)：主题化笔记与研究文档。
  - [`blockchain/`](../01-Notes/blockchain/)：区块链相关笔记
    - [`L2/`](../01-Notes/blockchain/L2/)、[`contract/`](../01-Notes/blockchain/contract/)、[`cross-chain/`](../01-Notes/blockchain/cross-chain/)、[`solana/`](../01-Notes/blockchain/solana/) 等子目录。
- [`02-Projects/`](../02-Projects/)：项目方案与设计文档（如 [`cosmos/`](../02-Projects/cosmos/)、[`hashkey/`](../02-Projects/hashkey/)、[`nesa/`](../02-Projects/nesa/)）。
- [`03-Resources/`](../03-Resources/)：资料与工具（例如 `软件.mdenc` 等）。
- [`Excalidraw/`](../Excalidraw/)：图形/草图文件。
- [`Templates/`](../Templates/)：写作与学习模板。

## 浏览建议

- 从 `01-Notes/` 进入具体主题，配合搜索快速定位；项目类内容从 `02-Projects/` 开始。
- 图示与草图在 `Excalidraw/`；图片素材统一放置在 `assets/images/`。
- 若使用站点预览，请以 `docs/` 为根进行导航；仓库内阅读以 `Home.md`/目录视图为入口。

说明：本首页仅描述结构组成，不列出具体内容细节；专题与技术条目请在对应目录中查阅。
